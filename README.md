# Experience Shop


## What is this?
* E-Shop.
* Project for SoftUni Angular 2020
* Simple and fast solution for people who search Web Store

## Features
* Authentication
* Share Announcements
* Buy & Search Functionality
* Delivery tracking
* etc. 

## Dependencies
* Angular
* Firebase
* Angular-Notifier
* Bootstrap
* Google Places

## How to run it?

```bash
    npm install

    npm serve
```
